# 安装

## 环境

>  [!Note]
>
>  要求Node版本v10.22以下，否则构建不成功，可到 [Node.js](https://nodejs.org) 下载并安装最新版本。

全局安装GitBook

```shell
npm install gitbook-cli -g
```

查看是否安装成功

```java
gitbook --version
```

初始化命令：创建文章

```shell
$ gitbook init
```

项目运行：本地启动服务

```shell
$ gitbook serve
```
