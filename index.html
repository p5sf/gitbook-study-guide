配置环境

Node 版本v10.22.0以下，否则构建不成功

# 全局安装GitBook
npm install gitbook-cli -g

安装插件并启动

$ gitbook install
$ gitbook serve

至此、可以用浏览器访问https://localhost:4000 查看书籍或者查看我的网上地址
